Module 14: String Formatting
Definition:

String formatting in Python allows you to create formatted strings using placeholders.

Basic Syntax:

# Example of string formatting:

Module 16: If Statement
Definition:

The if statement in Python is used to execute a block of code if a condition is true.

How to Use:

Use the if statement followed by a condition to execute a block of code if the condition evaluates to true.

Basic Syntax:

# Example of if statement:
x = 10
if x > 5:
  print("x is greater than 5")
Module 17: If-Else Statement
Definition:

The if-else statement in Python is used to execute one block of code if a condition is true and another block if the condition is false.

How to Use:

Use the if-else statement to execute different blocks of code based on whether a condition is true or false.

Basic Syntax:

# Example of if-else statement:
x = 10
if x > 5:
  print("x is greater than 5")
else:
  print("x is less than or equal to 5")
Module 18: If-Elif-Else Statement
Definition:

The if-elif-else statement in Python is used to execute one block of code among several alternatives.

How to Use:

Use the if-elif-else statement to choose among multiple blocks of code based on different conditions.

Basic Syntax:

# Example of if-elif-else statement:
x = 10
if x > 10:
  print("x is greater than 10")
elif x == 10:
  print("x is equal to 10")
else:
  print("x is less than 10")
Module 19: Nested If Statement
Definition:

Nested if statements in Python are if statements inside another if statement.

How to Use:

Use nested if statements to evaluate multiple conditions, where the inner if statement is executed only if the outer if statement's condition is true.

Basic Syntax:

# Example of nested if statement:
x = 10
if x > 5:
  print("x is greater than 5")
  if x == 10:
      print("x is equal to 10")
  else:
      print("x is not equal to 10")
else:
  print("x is less than or equal to 5")

Module 45: While Loop 5: Prime Numbers

num = 2

while num <= n:
  prime = True
  divisor = 2
  while divisor <= num / 2:

          prime = False
          break
      divisor += 1
  if prime:
      print(num, end=' ')
  num += 1
# Output:
# 2 3 5 7

Module 99: Stacks
Definition:

A stack is a data structure that follows the Last In, First Out (LIFO) principle, where the last element added is the first one to be removed. Stacks are commonly used in various applications such as expression evaluation, function call management, and backtracking algorithms.

Scenario:

Illustrate how to implement and use a stack in Python. Show how to perform basic stack operations such as push (addition), pop (removal), and peek (accessing the top element), as well as checking for empty and full conditions.

Examples:

# Example of implementing a stack in Python
class Stack:
  def __init__(self):
      self.items = []

  def is_empty(self):
      return self.items == []

  def push(self, item):
      self.items.append(item)

  def pop(self):
      if not self.is_empty():
          return self.items.pop()

  def peek(self):
      if not self.is_empty():
          return self.items[-1]

  def size(self):
      return len(self.items)

# Using the stack
stack = Stack()
stack.push(1)
stack.push(2)
stack.push(3)
print(stack.peek())  # Output: 3
print(stack.pop())   # Output: 3
print(stack.size())  # Output: 2

Module 102: Linked Lists
Definition:

A linked list is a linear data structure consisting of a sequence of elements where each element points to the next one in the sequence. Linked lists offer efficient insertion and deletion operations compared to arrays, but have slower access times due to their sequential nature.

Scenario:

Demonstrate how to implement and use linked lists in Python. Show how to perform basic linked list operations such as insertion, deletion, traversal, and searching, as well as illustrating different types of linked lists such as singly linked lists, doubly linked lists, and circular linked lists.

Examples:

# Example of implementing a singly linked list in Python
class Node:
  def __init__(self, data):
      self.data = data
      self.next = None

class LinkedList:
  def __init__(self):
      self.head = None

  def append(self, data):
      new_node = Node(data)
      if not self.head:
          self.head = new_node
          return
      current = self.head
      while current.next:
          current = current.next
      current.next = new_node

  def display(self):
      current = self.head
      while current:
          print(current.data, end=' -> ')
          current = current.next
      print('None')

# Using the singly linked list
ll = LinkedList()
ll.append(1)
ll.append(2)
ll.append(3)
ll.display()  # Output: 1 -> 2 -> 3 -> None
